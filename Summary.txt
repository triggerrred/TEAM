У цьому проєкті я навчився основам роботи з GitHub. Основні навички щодо керування версіями:
Клонування репозиторіїв:
Навчився клонувати репозиторій на локальний комп'ютер за допомогою команди git clone.
Створення гілок:
Засвоїв важливість створення гілок для роботи над новими функціями або виправленнями без впливу на основний код.
Використовував команду git checkout -b нова-функція.
Фіксація змін:
Навчився додавати зміни до індексу (git add .) і фіксувати їх із змістовним повідомленням (git commit -m "Додати нову функцію").
